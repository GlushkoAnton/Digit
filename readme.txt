

 Digit Recognizer – Прогнозирование цифр по изображениям

Проект представляет собой модель машинного обучения на базе сверточной нейронной сети (CNN), которая распознаёт рукописные цифры от 0 до 9 по их изображениям. Основан на датасете MNIST и реализован с использованием TensorFlow и Keras.

⸻

Технологии и библиотеки
	•	Python 3.x
	•	TensorFlow / Keras
	•	NumPy
	•	Pandas
	•	Matplotlib / Seaborn
	•	Scikit-learn

⸻

 Возможности
	•	Предобработка изображений (нормализация, one-hot encoding).
	•	Построение и обучение сверточной нейронной сети.
	•	Визуализация данных и матрицы ошибок.
	•	Прогноз класса (цифры) по изображению.
	•	Опциональное увеличение данных с помощью ImageDataGenerator.

⸻

 Как запустить
	1.	Установите зависимости:

pip install -r requirements.txt


	2.	Запустите Jupyter Notebook:

jupyter notebook


	3.	Откройте digit_recognizer.ipynb и следуйте шагам.

⸻

 Пример использования

predicted_label = model.predict(image)  # image размером 28x28


⸻

 Результаты

Модель достигает точности ~99% на валидации MNIST при корректной настройке архитектуры и гиперпараметров.
